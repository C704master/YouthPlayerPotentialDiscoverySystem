# 阶段 0 · 数据采集与位置补充（方案 B）

> 方案 B 落地新增的**数据采集层**。用 [ScraperFC](https://github.com/oseymour/ScraperFC) 抓取 FBref + Transfermarkt 数据，产出 `data/raw/players.csv`，供阶段 1 读取。
>
> 本文件是 docs 内的概要；**完整实现思路、伪代码、风险与合规见** [`F&Q/方案B-数据采集实现思路.md`](../F&Q/方案B-数据采集实现思路.md)。

## 阶段目标

用 ScraperFC 采集最新、最全、可复现的球员数据，并**补上细分位置**（Winger / AM / CM / DM），产出统一的 `data/raw/players.csv`。

## 输入 / 输出

| 项 | 内容 |
|----|------|
| 输入 | 联赛列表、赛季列表、`config/position_mapping.yaml` |
| 输出 | `data/raw/players.csv`（阶段 1 的输入） |
| 模块 | `collector.py` |

## 为什么用 ScraperFC 而非现成 CSV

- 覆盖 **10 联赛 × 3 赛季**（五大联赛 + 荷甲 + 葡超 + 英冠 + 欧冠 + 欧联，2023-24 ~ 2025-26），比单个 Kaggle 数据集更全、更新。
- 底层就是方案 B 封装 `football-data-mcp` 的数据来源，但我们直接用库拉成 CSV，符合"本地脚本 MVP"定位。
- 可复现：把采集脚本和结果 CSV 一并留存，答辩可展示完整数据管道。

## 采集要点

1. `pip install ScraperFC pandas pyarrow`
2. 用 `FBref(wait_time=7)` **限速**（FBref 每分钟 ≤10 请求）。
3. 每个联赛×赛季抓多个 stat 分类：`standard` + `passing` + `defense` + `possession` + `misc`，按 `player_id` 横向合并。
4. **两个坑**：FBref 是两级表头需扁平化；FBref 赛季表位置只有 FW/MF/DF，见下方位置补充。
5. 加 `league` 列并做**联赛名归一化**（FBref 的 `"EPL"` → Opta 强度表的 `"Premier League"`，供阶段 5）。
6. 落盘 `data/raw/players.csv` 并**提交一份缓存**，避免重复抓取。

## 位置细分补充（核心）

评分模型要把球员分到 Winger / AM / CM / DM（阶段 2/4），但 FBref 赛季表只给 FW/MF/DF。分层补充：

| 层 | 来源 | 说明 |
|----|------|------|
| 0 | FBref 粗位置 + 启发式 | MVP 先跑通，靠指标比值粗分，有误差 |
| **1（首选）** | **Transfermarkt 主位置** | `Right Winger→Winger`、`Attacking/Central/Defensive Midfield→AM/CM/DM`，**ScraperFC 同库自带**，命名近 1:1 |
| B（兜底） | FBref 球员个人页二级位置 | 解析 `Position: FW-MF (AM-CM-WM)`，只补 Top 20 候选，无新依赖 |
| 2（可选） | FotMob | 当季最新位置，需额外装库 |

映射配置 `config/position_mapping.yaml`：

```yaml
Right Winger: Winger
Left Winger: Winger
Winger: Winger
Attacking Midfield: AM
Central Midfield: CM
Defensive Midfield: DM
```

**跨源匹配难点**：FBref 与 Transfermarkt 无共享球员 ID，需按 `球员名 + 俱乐部` 模糊匹配（去重音、小写）；匹配不上回落第 0 层启发式并在数据完整度里标记。

## 验收标准

- [ ] `collector.py` 能产出 `data/raw/players.csv`，含 Age、Pos、Min、进攻/防守/过人等核心字段。
- [ ] 至少覆盖五大联赛 2024-25 完整赛季（主数据）。
- [ ] 大多数球员具备来自 Transfermarkt 的细分位置；未匹配上的有兜底与标记。
- [ ] `league` 列可映射到 Opta 联赛强度表。

## 风险提示（详见 F&Q）

- 赛季进行中：主榜用 2024-25 完整赛季，2025-26 单独出观察榜。
