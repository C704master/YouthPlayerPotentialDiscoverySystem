## 二、待改进（阶段 0-2 遗留问题）

### 2.1 数据字段不足（优先级：高）

当前数据集缺失 5 个核心评分字段（shots / dribbles / passes / tackles / interceptions），导致：
- DM 位置无法识别
- 阶段 3 特征计算缺少关键指标
- 数据完整度区分度不足

**解决方案**：
1. 补充含完整 FBref stat 分类的数据集（如 `emrey3lmaz/top-5-league-football-player-stats-2017-2025`，但需修复解析问题）
2. 或等 ScraperFC 适配 Python 3.14 后重新采集

### 2.2 位置来源单一（优先级：中）

当前全部为 heuristic，没有 Transfermarkt 数据。AM/CM 区分仅依赖 key_passes 比值，缺乏防守数据时 DM 无法识别。

**解决方案**：
1. 补充 Transfermarkt 爬虫（直接 request 到 transfermarkt.com，或使用 `transfermarkt-api` 包）
2. 或爬 FBref 球员个人页获取二级位置（方案 B 兜底层）

### 2.3 只覆盖 1 个赛季（优先级：低）

设计文档要求 3 个赛季（2023-24 ~ 2025-26），当前只有 2024-25。
