# 阶段 4 · 潜力评分模型

> 对应原文档：八、潜力评分模型（8.1 总体思路、8.2 分位置权重、8.3 百分位评分方法）／ 十六、风险与解决方案

## 阶段目标

能按 Winger / CM / AM / DM **分开**计算 0-100 分的潜力评分，评分必须透明、可解释、可复现。

潜力评分含义是「**当前阶段值得关注的程度**」，不是对未来成就的绝对预测。

## 输入 / 输出

| 项 | 内容 |
|----|------|
| 输入 | 阶段 3 的增强特征数据、`config/scoring_weights.yaml` |
| 输出 | `data/processed/scored_players.csv`（含总分和所有分项得分） |
| 模块 | `scorer.py` |

## 8.1 总体评分公式

```
最终潜力分 = 基础个人表现分 × 90% + 联赛强度分 × 10% - 风险扣分

其中：
基础个人表现分 = 年龄优势 + 出场可靠性 + 同位置表现 + 场上行为风格
联赛强度分     = 根据 Opta Power Rankings 转换得到的 0-100 分（见阶段 5）
风险扣分       = 红牌、黄牌过多、样本不足、数据完整度过低等因素
```

> **注意**：不建议直接使用「球员基础分 × 联赛系数」的方式，因为这样可能过度压低非五大联赛优秀年轻球员。联赛强度在第一版中只作为 10% 左右的轻量修正。

## 分项权重

| 分项 | 建议权重 | 解释 |
|------|---------|------|
| 年龄优势 | 15% | 越年轻且能获得稳定出场，说明稀缺性越高 |
| 出场可靠性 | 15% | 600 分钟以上才进入正式评分，分钟数越充分，数据越稳定 |
| 同位置核心表现 | 45% | 边锋、AM、CM、DM 分开比较，使用同位置百分位 |
| 场上行为风格 | 15% | 基于主动性、推进、创造、防守参与、纪律性等场上数据 |
| 联赛强度修正 | 10% | 参考 Opta Power Rankings，作为环境难度轻量修正（阶段 5） |
| 风险扣分 | 0 到 -10 分 | 红牌、黄牌、犯规、数据完整度低、样本不足等 |

## 8.2 分位置重点指标

| 位置 | 重点指标 | 设计理由 |
|------|---------|----------|
| **Winger 边锋** | 进球、助攻、射门、关键传球、过人、成功过人、传中、推进带球、禁区触球 | 边锋核心价值是制造进攻威胁和推进到危险区域 |
| **AM 前腰** | 关键传球、助攻、威胁传球、推进传球、传入禁区、射门创造动作 | 前腰重点是创造机会和连接进攻 |
| **CM 中前卫** | 传球、传球成功率、推进传球、回收球权、抢断、拦截、稳定性 | CM 需要兼顾组织、推进和攻防连接 |
| **DM 后腰** | 抢断、拦截、对抗成功率、回收球权、传球稳定、失误控制 | DM 重点是防守覆盖、球权保护和稳定出球 |

## 8.3 同位置百分位评分方法

第一版使用**同位置百分位排名**，而不是直接比较原始数据。这样可以避免不同位置职责不同导致的误判。

示例：

```
某 Winger 的 successful_dribbles_per90 在所有 Winger 中处于第 85 百分位
→ 该球员「突破能力」可以获得约 85 分

某 DM 的 tackles_per90 在所有 DM 中处于第 78 百分位
→ 该球员「防守覆盖」可以获得约 78 分
```

实现思路：先按 `standard_position` 分组，再对每个重点指标在组内计算百分位（0-100），加权得到该位置的「同位置核心表现分」。

## 实现要点

- `scorer.py` 按位置分组，分别计算各分项得分，再按权重合成。
- 年龄优势、出场可靠性可用简单归一化/分段函数。
- 风险扣分上限 -10（来自 `config.yaml` 的 `max_risk_penalty`）。
- 联赛强度分留出接口，由阶段 5 填入。
- 输出 `scored_players.csv`，包含 `total_score` 与所有分项列。

## 相关风险

| 风险 | 表现 | 解决方案 |
|------|------|----------|
| 小样本虚高 | 出场很少但 per90 很夸张 | 正式评分要求 minutes ≥ 600（阶段 2）+ 风险扣分 |
| 数据缺失 | 高级字段缺失 | 数据完整度过低参与风险扣分 |

## 验收标准

- [ ] 所有正式评分球员 `total_score` 在 0-100 之间。
- [ ] 评分结果中包含：年龄、出场、核心表现、行为风格、联赛强度、风险扣分等分项。

## 学习路径提示

- 学会把清洗、评分分别写成函数；理解「同位置百分位」的意义。
