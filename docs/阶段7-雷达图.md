# 阶段 7 · 雷达图

> 对应原文档：五、核心模块设计（chart_builder.py）／ 十一、输出文件设计（雷达图）

## 阶段目标

为 Top 20 每名球员生成一张能力结构**雷达图（PNG）**，直观展示分项能力。

## 输入 / 输出

| 项 | 内容 |
|----|------|
| 输入 | Top 20 球员的分项得分（阶段 4/6） |
| 输出 | `outputs/charts/player_name_radar.png`（每名球员一张） |
| 模块 | `chart_builder.py` |

## 输出物设计

| 输出物 | 格式 | 说明 |
|--------|------|------|
| 雷达图 | PNG | 每名 Top 20 球员一张 |

## 实现要点

- 使用 `matplotlib` 极坐标（polar）绘制雷达图。
- 雷达图的「轴」使用该球员位置的重点分项得分（来自阶段 4 的分项，如进攻威胁、创造、推进、防守参与、纪律性等）。
- 分数统一使用 0-100 量纲，保证不同球员图形可比。
- 文件命名与球员对应（`player_name_radar.png`），便于阶段 8 报告引用。
- 注意中文/特殊字符文件名与字体的兼容处理。

## 验收标准

- [ ] Top 20 每名球员至少生成一张 PNG 雷达图。

## 学习路径提示

- 学会 matplotlib 画雷达图（极坐标、闭合多边形、坐标轴标签）。
