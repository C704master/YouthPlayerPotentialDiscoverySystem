### 当前评分公式

```
total_score = 年龄优势×15% + 出场可靠性×15% + 同位置核心表现×45%
            + 行为风格×15% + 联赛强度(Opta)×10% - 风险扣分(0~-10)
```

import pandas as pd

print(top[['player_name','standard_position','league','age','total_score']])
"
